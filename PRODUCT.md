# Glance Deck Product Context

## Register

Product

## Product Purpose

Glance Deck brings the official Glance dashboard palettes to the Steam Deck through Decky CSS Loader. The theme should make SteamOS Gaming Mode feel cleaner, calmer, and more readable without fighting the native Steam Deck interface.

This is not a redesign of SteamOS. It is a restrained system-wide skin that tunes color, contrast, focus states, and corner radius while keeping the interaction model familiar.

## Users

- Steam Deck owners who use Decky CSS Loader and want a polished system-wide theme.
- Glance users who like its dashboard palettes and want the same mood on their handheld.
- Tinkerers who switch palettes often and expect configuration through CSS Loader rather than manual CSS edits.

## Usage Scene

A player is holding the Steam Deck at couch or desk distance, often under imperfect room lighting, moving quickly through the Quick Access Menu, Steam menu, library, settings, downloads, and game pages. Text must stay legible even when the font weight is light, the selected row is bright, or a panel is partially overlaying game artwork.

## Product Principles

- Readability first: palette fidelity is secondary to handheld contrast.
- Native-feeling controls: the UI should still feel like SteamOS, just cleaner.
- Configurable softness: rounded corners should be tunable, not imposed.
- No decorative interference: fades, glows, and shadows should never reduce legibility.
- Repeatable quality: palette changes should pass an automated contrast audit.

## Non Goals

- Replacing the SteamOS layout.
- Creating highly ornamental or animated UI.
- Styling every possible Steam selector at the cost of fragile CSS.
- Exact reproduction of Glance dashboard colors when those colors fail on Steam Deck surfaces.

## Tone

Quiet, practical, and friendly. The README can be warm and personal, but the theme itself should feel task-focused and trustworthy.
